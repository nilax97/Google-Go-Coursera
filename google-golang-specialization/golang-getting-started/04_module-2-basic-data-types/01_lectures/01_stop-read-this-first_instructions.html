Before watching the lectures, please note the following:

  1. Video Lecture M2.1.2: At 0:55, the instructor says "x=1" but should actually say "x=4." The screen text is correct.

  2. Video Lecture M2.2.2: Note that UTF-8 (Unicode Transformation Format - 8) is an encoding standard for Unicode along with UTF-16 and UTF-32. This latter standard is a fixed-width encoding, in which each 32-bit value represents one Unicode code point. Some programmers have UTF-32 in mind when they think about "Unicode." For more details about Unicode and UTF, please visit the FAQ page on the Unicode website: http://unicode.org/faq/utf_bom.html .
